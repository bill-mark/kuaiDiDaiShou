  #Ionic学习总结：
  1：该项目开发过程中，代码重构做的不够好，因为刚接触typescript和AngularJS ES6对其语法不够熟悉，导致代码略显臃肿，在后期的学习中需                  要改善。
  2：后期需要学习更多的cordova插件，比如：百度地图 支付宝微信支付等
  3：学习使用自定义开发插件



  #注意:
  该项目为window下开发，如在其他平台运行，需要重新ionic platform add  xxx,
  然后更具log对项目进行重构：ionic rebuild
